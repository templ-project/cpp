### Greeter

The main [Greeter](../cpp-template/classcpp__template_1_1_greeter.md) class provides greeting and farewell functionality:

- `Hello(const std::string& name)` - Creates a greeting message
- `Goodbye(const std::string& name)` - Creates a farewell message

### InvalidNameError

The [InvalidNameError](../cpp-template/classcpp__template_1_1_invalid_name_error.md) exception is thrown when invalid names are provided.
